## **INTRODUCCIÓN**

*Bienvenido a la documentación del Sistema de Gestión Académica, una aplicación móvil pensada para facilitar la organización de las actividades académicas de los estudiantes. El SGA permite gestionar tareas, horarios, calificaciones y recordatorios desde un solo lugar, ofreciendo un entorno intuitivo, accesible y eficiente.*

*Este repositorio proporciona una visión completa del proyecto, incluyendo su propósito, objetivos, alcance, funcionalidades, arquitectura y metodología de desarrollo. Además, detalla los requerimientos del sistema y los diagramas UML que respaldan su diseño, mostrando cómo cada componente contribuye a una experiencia de usuario organizada y confiable.*

*El SGA está diseñado para funcionar completamente offline, con almacenamiento seguro en SQLite, asegurando que los estudiantes puedan mantener el control de su vida académica en cualquier momento y lugar.*
